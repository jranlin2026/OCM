Q
极享AI 支持哪些平台的口播内容生成？
A

目前支持抖音、快手、视频号、小红书、B站、淘宝直播等主流短视频和直播平台。 系统会根据不同平台的算法偏好和内容风格，自动优化口播文案的语速、时长和关键词密度。 同时支持多账号统一管理和内容排期发布。

Q
899 标准版和代理版的主要区别是什么？
A

899 标准版面向个人创业者和单账号运营团队，提供基础的 AI 口播脚本生成和客户管理功能。 代理版则面向代理商和渠道伙伴，支持多客户池管理、分佣结算系统和团队协作空间。 简单来说，代理版是 899 的全功能升级版，增加了商业赋能工具。

Q
如何申请成为极享AI的代理商？
A

您可以通过以下方式申请成为代理商：1）在官网「招商代理」页面提交申请表单； 2）拨打招商热线 400-xxx-xxxx 联系渠道经理；3）通过已合作的代理商进行推荐。 审核周期一般为 3-5 个工作日，通过后享受专属渠道政策、培训体系和市场推广支持。

Q
AI 生成的脚本内容是否涉及版权问题？
A

极享AI 的大模型基于合法授权的训练数据，生成的脚本内容为原创输出，版权归属于使用方。 系统内置敏感词检测和合规审核机制，避免生成侵权或违规内容。 建议用户在发布前根据平台规则进行二次审校，确保内容合规。

版权 合规
Q
贴牌版支持哪些品牌定制功能？
A

贴牌版提供完整的品牌定制能力，包括：自定义品牌 Logo、品牌色系、域名绑定、 自有品牌 UI 界面、品牌水印、定制化短信/邮件通知模板等。 同时也支持 API 接口定制和功能模块的按需裁剪，满足不同场景的个性化需求。

Q
如何导出客户数据和订单记录？
A

在客户管理和订单管理模块中，均提供数据导出功能。支持导出格式包括：Excel（.xlsx）、 CSV（.csv）和 PDF 报表。您可以选择导出的字段范围和时间区间，导出结果将通过邮件发送 或在页面直接下载。高级版本支持自动化定期导出和 API 数据接口对接。

Q
AI 数字人播报需要额外付费吗？
A

数字人播报功能根据版本不同有所差异。899 标准版不含数字人功能； 代理版包含标准版数字人播报（基于模板形象）；贴牌版及以上版本支持自定义数字人形象克隆， 提供更丰富的表情动作和场景适配。具体价格请参考产品矩阵中心的对比表。
